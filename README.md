# 🐍 Python for Data Engineering

This repository documents my complete learning journey of Python from basics to advanced concepts, with a focus on building a strong foundation for Data Engineering.

---

## 🎯 Goal

- Master Python fundamentals
- Understand Object-Oriented Programming (OOP)
- Learn Data Structures & Algorithms (DSA)
- Apply concepts in real-world Data Engineering scenarios

---

## 🧭 Roadmap

### 🟢 1. Programming Fundamentals (PF)
- Variables & Data Types
- Input/Output
- Conditional Statements
- Loops
- Functions
- Lists, Tuples, Dictionaries
- File Handling

---

### 🟡 2. Object-Oriented Programming (OOP)
- Classes & Objects
- Constructors
- Inheritance
- Encapsulation
- Polymorphism

---

### 🔵 3. Data Structures & Algorithms (DSA)
- Arrays / Lists
- Strings
- Stacks & Queues
- HashMaps (Dictionaries)
- Recursion
- Sorting & Searching

---

### 🟣 4. Data Engineering Foundations
- SQL Basics
- Data Processing Concepts
- ETL (Extract, Transform, Load)
- Working with structured data
- Introduction to tools like Apache Spark
